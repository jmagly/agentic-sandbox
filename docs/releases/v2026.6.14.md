## What This Release Is

v2026.6.14 is the direct-delivery CalVer release-flow cut for the local-first
secure agent transport work. It includes the local CA backend lifecycle,
remote CA backend boundary, deterministic CA renewal test, target-typed PTY
ioctl fix, mutsu Darwin smoke restructuring, and final publication hardening
for SBOM upload plus GitHub release mirroring.

## Highlights

- Darwin binaries execute on mutsu in the native build session and the final
  Darwin tarball is verified locally for arm64 Mach-O payloads.
- SBOM generation uploads are hardened against syft `/tmp` scratch pressure.
- The GitHub release mirror secret has been refreshed with a `repo` scoped
  token for the required public release sync.
