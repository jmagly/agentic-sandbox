## What this release is

v2026.6.8 is the release-attempt replacement for `v2026.6.7`. It preserves the
CA backend lifecycle and direct-release-flow content, and fixes the mutsu SSH
build lane that failed to find `rustup` during Apple Silicon release artifact
publication.

## Highlights

- **CA backend lifecycle** (#492/#493): same local-first gRPC mTLS CA backend,
  fail-closed remote boundary, and bootstrap CSR enrollment behavior introduced
  in `v2026.6.7`.
- **Mutsu Rust tooling resolution** (#481): the SSH build now expands `PATH`
  across the external cargo home, the automation user's cargo home, Homebrew,
  and the cached stable toolchain path before invoking Rust tools.
- **Clear mutsu diagnostics**: the release job prints resolved `rustup`,
  `cargo`, `rustc`, and version information before building; if `rustup` is not
  available, it only proceeds when `rustc` can already resolve the target
  standard library.
