## What this release is

A CI/docs release. The runtime is **identical** to v2026.7.10 — same binaries,
same behavior. This bump re-runs the full release pipeline to verify it end to
end now that CI secrets come entirely from OpenBao and non-secret config comes
from Gitea variables, and it adds automatic Cloudflare cache purging when docs
are published.

## Highlights

- **Docs publish now busts the cache.** After deploying docs to
  `docs.aiwg.io/agentic-sandbox/`, CI fetches the Cloudflare token from OpenBao
  and purges the subpath (scoped to the tenant, batched 30/request) so readers
  get new docs immediately instead of waiting on the cache TTL.
- **Non-secret CI config → Gitea variables.** `DEPLOY_HOST/PATH/PORT/USER` are
  now repository variables; only secret-zero (`BAO_*`), the Cloudflare zone id,
  and vault-sourced key material remain Actions secrets.
- **Pure-vault verification.** Every publication path — internal registry, GHCR
  mirror, GitHub mirror, docsite deploy + purge, and GPG tarball signing — runs
  on the OpenBao AppRole with nothing sensitive left in the tracker.
